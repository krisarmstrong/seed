## Critical Gaps We MUST Address

### **MUST HAVE** (Launch Blockers)

1. **RF Path Loss Modeling (Issue #591)**
   - **Current:** Basic Free Space Path Loss (FSPL)
   - **Needed:** Log-distance path loss + wall attenuation + multi-wall model
   - **Accuracy target:** ±10 dB (competitive with Hamina)
   - **Priority:** CRITICAL - Without this, predictive survey is toy

2. **Heat Map Generation (Issue #588)**
   - **Current:** Planned
   - **Needed:** Smooth interpolation (IDW or Kriging), color-coded
   - **Visual quality:** Must look professional, not amateur
   - **Priority:** CRITICAL - This is baseline expectation

3. **Floor Plan Editor**
   - **Current:** Not implemented
   - **Needed:** Draw walls, set materials, scale floor plan
   - **UX:** Must be intuitive (click-drag interface)
   - **Priority:** CRITICAL - Can't do predictive survey without this

4. **AP Database**
   - **Current:** Not implemented
   - **Needed:** Database of AP models with TX power, antenna gain, etc.
   - **Coverage:** Ubiquiti, Aruba, Cisco, Ruckus, TP-Link at minimum
   - **Priority:** HIGH - Needed for accurate predictions

5. **WiFi 6/6E Support**
   - **Current:** Not implemented
   - **Needed:** 2.4 GHz, 5 GHz, 6 GHz bands
   - **Priority:** HIGH - WiFi 6E is becoming standard

### **SHOULD HAVE** (Competitive Parity)

1. **Active Survey Mode**
   - **Current:** Basic WiFi scanning
   - **Needed:** Walk-around survey with GPS/manual points, continuous scanning
   - **Priority:** MEDIUM - Hamina doesn't have this, differentiator vs them

2. **Passive Survey Mode**
   - **Current:** Basic WiFi scanning
   - **Needed:** Monitor mode, all-channel scanning, BSSID tracking
   - **Priority:** MEDIUM - Required for post-deployment validation

3. **Multi-Floor Support**
   - **Current:** Single floor
   - **Needed:** Multiple floors with floor plan stacking
   - **Priority:** MEDIUM - Hospitals, offices need this

### **NICE TO HAVE** (Future Differentiation)

1. **Client Density Modeling**
   - Predict max concurrent clients per AP
   - Required for stadiums, conference centers

2. **Throughput Prediction**
   - Estimate data rates based on RSSI and MCS
   - "This location will get 50 Mbps"

3. **Cost Optimization**
   - "Use 3 high-power APs instead of 5 low-power" → save $400
