Installation Complete!

The Seed has been installed and the service is starting.

Next Steps:

  1. Open your browser and navigate to:
    https://localhost:8443
  2. If this is a fresh install, you'll be prompted to set up your admin password.
  3. Accept the self-signed certificate warning to access the web interface.

Useful Commands:

View logs:

tail -f /usr/local/seed/logs/seed.log

Stop service:

sudo launchctl unload /Library/LaunchDaemons/com.seed.plist

Start service:

sudo launchctl load /Library/LaunchDaemons/com.seed.plist
Note: The Seed runs as a background service and will start automatically at system boot.
