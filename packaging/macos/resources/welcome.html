Welcome to The Seed

The Seed is a comprehensive network diagnostic tool for network professionals and enthusiasts.

What will be installed:

  • The Seed binary at /usr/local/seed/
  • Launchd service for automatic startup
  • Web interface accessible at https://localhost:8443

Key Features:

  • Real-time link monitoring
  • Gateway and DNS testing
  • Network discovery (LLDP/CDP/EDP)
  • WiFi analysis and channel scanning
  • Performance testing
Note: This installer requires administrator privileges to install system-level components and configure the launchd service.
